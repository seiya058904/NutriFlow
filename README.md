Minimal Daily Nutrition & Weight Tracking

极简每日饮食与体重趋势记录工具

## 使用方法

下载 **`NutriFlow.html`** 这一个文件，双击在浏览器中打开即可使用。

> **注意：** 请不要下载 `index.html`，该文件包含开发者测试数据。只需要 `NutriFlow.html` 即可。

数据保存在浏览器本地（localStorage），无需服务器，刷新页面数据仍在。
